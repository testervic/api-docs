# 更新日志

## v1.2.0 — 2025-06-01

### 新增
- 资源接口支持按标签筛选（`/v1/resources?tag=xxx`）
- 用户列表接口支持按角色筛选

### 修复
- 修复 Token 刷新后旧 Token 未立即失效的问题

---

## v1.1.0 — 2025-03-15

### 新增
- 新增 `PATCH /v1/resources/{id}` 部分更新接口
- 错误响应新增 `request_id` 字段，便于问题排查

### 变更
- 列表接口分页参数统一改为 `page` + `per_page`（原 `offset` + `limit` 仍兼容）

- 正式发布
