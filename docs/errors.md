# 错误码

当请求失败时，响应体中 `code` 字段为非零值，`message` 字段包含错误描述。

**错误响应示例：**

```json
{
  "code": 40100,
  "message": "未授权，请先登录",
  "data": null
}
```

## HTTP 状态码

| HTTP 状态码 | 说明 |
|-------------|------|
| `200` | 请求成功 |
| `400` | 请求参数错误 |
| `401` | 未授权 |
| `403` | 无权限 |
| `404` | 资源不存在 |
| `429` | 请求过于频繁 |
| `500` | 服务器内部错误 |

## 业务错误码

### 通用

| 错误码 | 说明 |
|--------|------|
| `40000` | 请求参数错误 |
| `40001` | 必填字段缺失 |
| `40002` | 字段格式不正确 |
| `40003` | 字段值超出范围 |

### 认证

| 错误码 | 说明 |
|--------|------|
| `40100` | 未授权，Token 缺失 |
| `40101` | Token 无效或已过期 |
| `40102` | Token 已被吊销 |
| `40300` | 无权限执行此操作 |

### 用户

| 错误码 | 说明 |
|--------|------|
| `40410` | 用户不存在 |
| `40910` | 邮箱已被注册 |

### 资源

| 错误码 | 说明 |
|--------|------|
| `40420` | 资源不存在 |
| `40320` | 无权访问此资源 |

### 服务端

| 错误码 | 说明 |
|--------|------|
| `50000` | 服务器内部错误 |
| `50300` | 服务暂时不可用，请稍后重试 |
